# Collaborative Editor - Backend

This is the Node.js backend for the Collaborative Real-Time Code Editor.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
   *The server runs on port 5000 by default.*

## 🛠️ Features

- **Socket.io Server**: Handles WebSocket connections for real-time communication.
- **YSocketIO**: Manages Yjs document synchronization and awareness states.
- **REST Endpoints**: Includes a health check endpoint at `/health`.

For full project documentation and setup instructions, please refer to the [Root README](../README.md).
