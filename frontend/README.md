# Collaborative Editor - Frontend

This is the React-based frontend for the Collaborative Real-Time Code Editor.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Key Components

- **App.jsx**: Main application logic, handles Yjs document initialization and Socket.io provider.
- **Monaco Editor**: Integrated using `@monaco-editor/react`.
- **Synchronization**: Uses `y-monaco` to bind Yjs types to the Monaco editor instance.

For full project documentation and setup instructions, please refer to the [Root README](../README.md).
